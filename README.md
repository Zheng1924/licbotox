# Long Island City Botox Website

A premium beauty services website built with Next.js, Tailwind CSS, and Framer Motion.

## Deploy to Vercel

### Option 1: Vercel CLI
```bash
npx vercel --prod
```

### Option 2: GitHub + Vercel Dashboard
1. Push this repo to GitHub
2. Go to vercel.com
3. Import from GitHub
4. Click Deploy

## Development
```bash
npm install
npm run dev
```
